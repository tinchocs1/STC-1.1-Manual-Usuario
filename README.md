# STC-1.1 | Sistema Térmico de Corte Controlado

Este repositorio contiene la documentación técnica y el manual de usuario para la construcción del **STC-1.1**, un dispositivo de código abierto diseñado para optimizar el proceso de corte simultáneo de envases de vidrio hueco.

## 📋 Sobre el Proyecto
El **STC-1.1** surge como Trabajo de Grado en la **Escuela Universitaria Centro de Diseño (EUCD | FADU | Udelar)**. Su objetivo es democratizar el acceso a herramientas de producción eficientes para emprendedores dedicados a la economía circular y el reciclaje de vidrio en Uruguay.

### Características principales:
- **Corte simultáneo:** Capacidad para procesar múltiples botellas a la vez.
- **Código Abierto:** Diseño modular y "hackeable" para adaptarlo a diferentes necesidades.
- **Eficiencia Térmica:** Sistema basado en resistencias de cromo-níquel con control de temperatura (50-300°C).

## 🛠️ Especificaciones Técnicas
- **Motores:** Sincrónicos de rotación constante.
- **Estructura:** Sistema de altura ajustable para diversos diámetros de envases.
- **Seguridad:** Aislamiento térmico mediante cableado de tela y señalización lumínica.
